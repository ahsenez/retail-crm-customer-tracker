# Retail CRM Customer Tracker

## Özellikler
- Müşteri ekleme
- Müşteri listeleme
- Segment bazlı filtreleme
- Son görüşme notu ekleme
- CSV dosyasına kaydetme
- Basit satış/CRM mantığı

## Kullanılan Teknolojiler
- Python
- CSV veri saklama
- Komut satırı uygulaması

## Kurulum
```bash
python main.py
```

git init
